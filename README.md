# AI Question Generator

An intelligent tool for automatically generating diverse and contextual questions using AI models. This project leverages state-of-the-art language models to create high-quality questions for various applications.

## 📋 Overview

The AI Question Generator is built as a Jupyter Notebook that demonstrates how to use modern AI/ML techniques to generate meaningful and diverse questions. Whether you need questions for educational purposes, content creation, or data augmentation, this tool can help automate the process.

## 🚀 Features

- **AI-Powered Generation**: Uses advanced language models to create contextually relevant questions
- **Batch Processing**: Handles multiple inputs efficiently
- **Customizable Parameters**: Fine-tune question generation to match your specific needs
- **Interactive Interface**: Built in Jupyter Notebook for easy experimentation and iteration
- **GPU Acceleration**: Configured for T4 GPU support in Google Colab for faster processing

## 📦 Requirements

- Python 3.x
- Jupyter Notebook or Google Colab
- Required Python packages (see notebook for details)
- GPU support (recommended for faster generation)
